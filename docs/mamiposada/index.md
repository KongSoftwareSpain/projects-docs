# POSMamiPosada TPV

Aplicación de **Terminal Punto de Venta (TPV)** desarrollada en **.NET 8 / WPF** para la gestión comercial de MamiPosada.

## Contenido

- **Onboarding** — Introducción al proyecto y configuración del entorno de desarrollo.
- **Arquitectura** — Visión general del sistema, patrones y decisiones técnicas.
- **Guías** — Convenciones de código y buenas prácticas del equipo.
- **Otros** — Recursos, enlaces y material complementario.
