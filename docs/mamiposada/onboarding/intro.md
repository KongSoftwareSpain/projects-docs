# Onboarding — POSMamiPosada

Objetivo: que cualquier desarrollador nuevo pueda arrancar y contribuir al proyecto sin necesidad de asistencia.

## ¿Qué es POSMamiPosada?

POSMamiPosada es un **Terminal Punto de Venta (TPV)** de escritorio desarrollado por **Kong Software** para la cadena MamiPosada. Permite gestionar ventas, pedidos, clientes, stock, impresión de tickets y etiquetas, control de turnos y más.

## Stack tecnológico

| Capa | Tecnología |
|---|---|
| Framework | .NET 8 (Windows) |
| UI | WPF (XAML) |
| Patrón | MVVM (Model-View-ViewModel) |
| Base de datos | Microsoft Access (`.mdb`) vía OLE DB |
| Plataforma objetivo | x86 |
| Librerías externas | PdfSharpCore, EPPlus, BarcodeLib, BarcodeStandard, Microsoft.Xaml.Behaviors.Wpf |
